PixbufSimpleAnim

open class PixbufSimpleAnim : PixbufAnimation, PixbufSimpleAnimProtocol

The PixbufSimpleAnim type acts as a reference-counted owner of an underlying GdkPixbufSimpleAnim instance. It provides the methods that can operate on this data type through PixbufSimpleAnimProtocol conformance. Use PixbufSimpleAnim as a strong reference or owner of a GdkPixbufSimpleAnim instance.

An opaque struct representing a simple animation.
